## What this scanner is doing (in one line)

It builds a **deterministic “change ledger”**:

## Your objective, translated

Input:
- current filesystem (`KDB/raw/**`)
- previous known state (`manifest.json.sources`)

Output:
- `KDB/state/last_scan.json` containing:
  - per-file classification (`NEW`, `CHANGED`, `UNCHANGED`, `MOVED`, `DELETED`, `ERROR`)
  - reconcile operations (`MOVED`, `DELETED`) for downstream logic

This output is the **single source of truth** for the compiler run.

---

## Why each invariant matters

1. **Hash authoritative; mtime advisory**
   - mtime can lie/noise (sync tools, copy ops)
   - hash tells true content equality

2. **Skip symlinks**
   - avoids loops, broken links, non-local surprises
   - keeps scanner deterministic

3. **Classify binaries**
   - scanner still tracks them
   - compiler can later do metadata-only or specialized parser path

4. **Two-pass rename detection**
   - critical to avoid false `NEW + DELETED` for same moved file
   - match by hash first, then classify leftovers

5. **Atomic write**
   - prevents partial/corrupt `last_scan.json`
   - especially important on synced folders (OneDrive)

6. **Pure function shape**
   - reproducible, testable, debuggable
   - no hidden state mutations
   - only side-effect: one output file write

7. **Typed outputs (`ScanEntry`, `ReconcileOp`, etc.)**
   - makes downstream planner/compiler/updater predictable
   - catches schema drift early

## Recommended scan algorithm (mental model)

### Phase A: Collect current raw snapshot
For each real file in `KDB/raw/`:
- path
- sha256 hash
- mtime
- size
- type/binary flag

Also collect skipped entries (symlinks, unreadable files).

### Phase B: Path intersection compare
For paths present in both current and manifest:
- same hash → `UNCHANGED`
- different hash → `CHANGED`

### Phase C: Rename/move matching (critical pass)
For paths missing from manifest side vs current side:
- hash-match old→new => `MOVED` reconcile op
- remove matched pairs from unmatched sets

### Phase D: Remaining unmatched
- current-only leftover => `NEW`
- manifest-only leftover => `DELETED` reconcile op

### Phase E: Emit `ScanResult`
- summary counts
- `entries[]`
- `to_compile[]` (NEW + CHANGED)
- `to_reconcile[]` (MOVED + DELETED)
- `to_skip[]` (UNCHANGED)
- skipped/errors lists
- atomic write once

---

## What downstream should assume

- Planner trusts `to_compile` and `to_reconcile`
- Compiler never re-decides what changed
- Patch applier only applies planned changes
- Manifest updater only applies this run’s result
- No component recomputes filesystem diff independently

That keeps the pipeline coherent.

## Quick sanity checks you should enforce in tests

- Move file without edits → exactly 1 `MOVED`, no NEW/DELETED pair
- Copy file (same hash both places) + keep old path → should be `NEW` (not moved)
- Edit file content same path → `CHANGED`
- Touch file only (mtime changes, hash same) → `UNCHANGED`
- Delete file → `DELETED`
- Symlink in raw → appears in skipped, not in entries

### Why not C

## Q1. Binary detection
My call: **extension + first-512-byte or first-4KB NUL sniff**.

### Why
Extension-only is too weak:
- people misname files
- copied exports often have weird extensions
- `.md` can contain binary junk if corrupted

NUL sniff alone is also not enough for classification ergonomics.

### Best practical rule
Use:
1. extension hint
2. binary sniff on first chunk
3. mark `is_binary: true/false`

Not:
- allowlist that excludes everything except `.md` / `.txt`

Because your raw layer may eventually include:
- `.pdf`
- `.json`
- `.html`
- `.csv`
- `.docx`
- images

The scanner should **track** them even if the compiler later does `metadata_only`.

So:
- scanner = broad
- compiler = selective

That separation is healthy.

---

## Q2. Rename detection threshold
My call: **pure hash equality only**.

### Why
The scanner is the pipeline’s truth source.  
Truth source should be conservative and high-confidence.

If you allow “probable rename” based on:
- same size
- close mtime

you will create false positives, and those are much worse than false negatives here.

### Better pattern
Use:
- **definite move** = exact same hash
- otherwise:
  - old file = `DELETED`
  - new file = `NEW`

If later you want heuristic assistance, add it as a separate non-authoritative field like:

```json
"probable_matches": [...]
```

But do **not** let that drive state reconciliation in v1.

But the canonical form should stay explicit.

```python name=check_last_scan_semantics.py
import json
import sys
from collections import Counter
from pathlib import Path


VALID_FILE_STATUSES = {"NEW", "CHANGED", "UNCHANGED", "MOVED", "DELETED"}
VALID_RECONCILE_TYPES = {"MOVED", "DELETED"}


def load_json(path: Path):
    return json.loads(path.read_text(encoding="utf-8"))


def main():
    root = Path.cwd()
    data_path = root / "KDB" / "state" / "last_scan.json"

    if not data_path.exists():
        print(f"ERROR: Missing data file: {data_path}")
        sys.exit(2)

    try:
        data = load_json(data_path)
    except Exception as e:
        print(f"ERROR: Failed reading JSON: {e}")
        sys.exit(2)

    errors = []

    files = data.get("files", [])
    to_compile = data.get("to_compile", [])
    to_skip = data.get("to_skip", [])
    to_reconcile = data.get("to_reconcile", [])
    summary = data.get("summary", {})
    scan_errors = data.get("errors", [])
    skipped_symlinks = data.get("skipped_symlinks", [])

    # --------------------------------------------------
    # Build indexes
    # --------------------------------------------------
    file_paths = []
    status_by_path = {}
    moved_targets = set()
    moved_sources = set()
    reconcile_moved_from = set()
    reconcile_moved_to = set()
    reconcile_deleted_paths = set()

    status_counter = Counter()

    for entry in files:
        path = entry.get("path")
        status = entry.get("status")

        if not path:
            errors.append("files[] contains entry with missing path")
            continue
        if status not in VALID_FILE_STATUSES:
            errors.append(f"files[{path}] has invalid status: {status}")
            continue

        file_paths.append(path)

        if path in status_by_path:
            errors.append(f"Duplicate files[] path detected: {path}")
        status_by_path[path] = status
        status_counter[status] += 1

        if status == "MOVED":
            moved_sources.add(path)
            moved_to = entry.get("moved_to")
            if not moved_to:
                errors.append(f"MOVED file entry missing moved_to: {path}")
            else:
                moved_targets.add(moved_to)

    # --------------------------------------------------
    # Basic uniqueness checks
    # --------------------------------------------------
    if len(file_paths) != len(set(file_paths)):
        errors.append("files[] contains duplicate path values")

    if len(to_compile) != len(set(to_compile)):
        errors.append("to_compile contains duplicate paths")

    if len(to_skip) != len(set(to_skip)):
        errors.append("to_skip contains duplicate paths")

    # --------------------------------------------------
    # to_compile consistency
    # --------------------------------------------------
    for path in to_compile:
        if path not in status_by_path:
            errors.append(f"to_compile path not found in files[]: {path}")
        else:
            if status_by_path[path] not in {"NEW", "CHANGED"}:
                errors.append(
                    f"to_compile path has invalid status {status_by_path[path]} (expected NEW/CHANGED): {path}"
                )

    # --------------------------------------------------
    # to_skip consistency
    # --------------------------------------------------
    for path in to_skip:
        if path not in status_by_path:
            errors.append(f"to_skip path not found in files[]: {path}")
        else:
            if status_by_path[path] != "UNCHANGED":
                errors.append(
                    f"to_skip path has invalid status {status_by_path[path]} (expected UNCHANGED): {path}"
                )

    # --------------------------------------------------
    # Reconcile consistency
    # --------------------------------------------------
    for op in to_reconcile:
        typ = op.get("type")
        if typ not in VALID_RECONCILE_TYPES:
            errors.append(f"Invalid reconcile op type: {typ}")
            continue

        if typ == "MOVED":
            src = op.get("from")
            dst = op.get("to")
            if not src or not dst:
                errors.append(f"MOVED reconcile op missing from/to: {op}")
                continue

            reconcile_moved_from.add(src)
            reconcile_moved_to.add(dst)

            # files[] should include a MOVED entry at old path
            if src not in status_by_path:
                errors.append(f"MOVED reconcile source not found in files[]: {src}")
            elif status_by_path[src] != "MOVED":
                errors.append(
                    f"MOVED reconcile source has status {status_by_path[src]} instead of MOVED: {src}"
                )

            # dst should not also appear as NEW in files[] if move was already reconciled
            if dst in status_by_path and status_by_path[dst] == "NEW":
                errors.append(
                    f"Rename target double-counted as NEW despite MOVED reconcile: {dst}"
                )

        elif typ == "DELETED":
            path = op.get("path")
            if not path:
                errors.append(f"DELETED reconcile op missing path: {op}")
                continue

            reconcile_deleted_paths.add(path)

            if path not in status_by_path:
                errors.append(f"DELETED reconcile path not found in files[]: {path}")
            elif status_by_path[path] != "DELETED":
                errors.append(
                    f"DELETED reconcile path has status {status_by_path[path]} instead of DELETED: {path}"
                )

    # --------------------------------------------------
    # MOVED file entries should have matching reconcile ops
    # --------------------------------------------------
    for src in moved_sources:
        if src not in reconcile_moved_from:
            errors.append(f"MOVED file entry missing corresponding reconcile op: {src}")

    for src in reconcile_moved_from:
        if src not in moved_sources:
            errors.append(f"MOVED reconcile op missing corresponding files[] entry: {src}")

    # --------------------------------------------------
    # DELETED file entries should have matching reconcile ops
    # --------------------------------------------------
    deleted_paths = {p for p, s in status_by_path.items() if s == "DELETED"}
    for path in deleted_paths:
        if path not in reconcile_deleted_paths:
            errors.append(f"DELETED file entry missing corresponding reconcile op: {path}")

    for path in reconcile_deleted_paths:
        if path not in deleted_paths:
            errors.append(f"DELETED reconcile op missing corresponding files[] entry: {path}")

    # --------------------------------------------------
    # Disjointness checks
    # --------------------------------------------------
    overlap_compile_skip = set(to_compile) & set(to_skip)
    if overlap_compile_skip:
        errors.append(f"Paths appear in both to_compile and to_skip: {sorted(overlap_compile_skip)}")

    # Paths marked MOVED or DELETED should not appear in to_compile / to_skip
    forbidden = moved_sources | deleted_paths
    overlap_compile_forbidden = forbidden & set(to_compile)
    overlap_skip_forbidden = forbidden & set(to_skip)

    if overlap_compile_forbidden:
        errors.append(f"MOVED/DELETED paths illegally appear in to_compile: {sorted(overlap_compile_forbidden)}")
    if overlap_skip_forbidden:
        errors.append(f"MOVED/DELETED paths illegally appear in to_skip: {sorted(overlap_skip_forbidden)}")

    # --------------------------------------------------
    # Summary count consistency
    # --------------------------------------------------
    expected_summary = {
        "new": status_counter["NEW"],
        "changed": status_counter["CHANGED"],
        "unchanged": status_counter["UNCHANGED"],
        "moved": status_counter["MOVED"],
        "deleted": status_counter["DELETED"],
        "error": len(scan_errors),
        "skipped_symlink": len(skipped_symlinks),
    }

    for key, expected in expected_summary.items():
        actual = summary.get(key)
        if actual != expected:
            errors.append(
                f"summary.{key}={actual} does not match computed value {expected}"
            )

    # --------------------------------------------------
    # Stronger cardinality sanity checks
    # --------------------------------------------------
    expected_to_compile = sorted(
        [p for p, s in status_by_path.items() if s in {"NEW", "CHANGED"}]
    )
    if sorted(to_compile) != expected_to_compile:
        errors.append(
            f"to_compile does not exactly match NEW+CHANGED files. "
            f"expected={expected_to_compile} actual={sorted(to_compile)}"
        )

    expected_to_skip = sorted(
        [p for p, s in status_by_path.items() if s == "UNCHANGED"]
    )
    if sorted(to_skip) != expected_to_skip:
        errors.append(
            f"to_skip does not exactly match UNCHANGED files. "
            f"expected={expected_to_skip} actual={sorted(to_skip)}"
        )

    expected_reconcile_count = status_counter["MOVED"] + status_counter["DELETED"]
    if len(to_reconcile) != expected_reconcile_count:
        errors.append(
            f"to_reconcile length {len(to_reconcile)} does not match MOVED+DELETED count {expected_reconcile_count}"
        )

    # --------------------------------------------------
    # Final result
    # --------------------------------------------------
    if errors:
        print("INVALID: last_scan.json failed semantic checks")
        for i, err in enumerate(errors, 1):
            print(f"{i}. {err}")
        sys.exit(1)

    print("OK: last_scan.json passed semantic checks")
    sys.exit(0)


if __name__ == "__main__":
    main()
```

```python name=check_compile_result_semantics.py
#!/usr/bin/env python3
"""
Semantic validator for KDB/state/compile_result.json

Checks cross-field invariants beyond JSON Schema.

Exit codes:
- 0: valid
- 1: semantic errors found
- 2: runtime/config error
"""

import json
import sys
from pathlib import Path


VALID_PAGE_TYPES = {"summary", "concept", "article", "index", "log"}
VALID_PAGE_STATUS = {"active", "stale", "orphan_candidate", "archived"}
VALID_CONFIDENCE = {"low", "medium", "high"}
VALID_SOURCE_REF_ROLES = {"primary", "supporting", "historical"}


def load_json(path: Path):
    return json.loads(path.read_text(encoding="utf-8"))


def normalize_page_type_from_path(page_id: str):
    p = page_id.replace("\\", "/")
    if p.endswith("/index.md") or p.endswith("wiki/index.md"):
        return "index"
    if p.endswith("/log.md") or p.endswith("wiki/log.md"):
        return "log"
    if "/summaries/" in p:
        return "summary"
    if "/concepts/" in p:
        return "concept"
    if "/articles/" in p:
        return "article"
    return None


def main():
    root = Path.cwd()
    data_path = root / "KDB" / "state" / "compile_result.json"

    if not data_path.exists():
        print(f"ERROR: Missing data file: {data_path}")
        sys.exit(2)

    try:
        data = load_json(data_path)
    except Exception as e:
        print(f"ERROR: Failed reading JSON: {e}")
        sys.exit(2)

    errors = []

    run_id = data.get("run_id")
    success = data.get("success")
    compiled_sources = data.get("compiled_sources", [])

    if not isinstance(run_id, str) or not run_id.strip():
        errors.append("run_id must be a non-empty string")

    if not isinstance(success, bool):
        errors.append("success must be a boolean")

    # --------------------------------------------------
    # Top-level uniqueness
    # --------------------------------------------------
    source_ids = []
    page_ids_global = []

    for cs in compiled_sources:
        sid = cs.get("source_id")
        if sid:
            source_ids.append(sid)

        for p in cs.get("pages", []):
            pid = p.get("page_id")
            if pid:
                page_ids_global.append(pid)

    if len(source_ids) != len(set(source_ids)):
        errors.append("compiled_sources contains duplicate source_id values")

    # Global duplicate pages across sources are risky in one compile_result;
    # force one owner/update record per page for v1 simplicity.
    if len(page_ids_global) != len(set(page_ids_global)):
        dupes = sorted({x for x in page_ids_global if page_ids_global.count(x) > 1})
        errors.append(f"compile_result pages contain duplicate page_id across compiled_sources: {dupes}")

    # --------------------------------------------------
    # Per-source semantic checks
    # --------------------------------------------------
    for idx, cs in enumerate(compiled_sources):
        prefix = f"compiled_sources[{idx}]"

        source_id = cs.get("source_id")
        summary_page = cs.get("summary_page")
        outputs_created = cs.get("outputs_created", [])
        outputs_touched = cs.get("outputs_touched", [])
        concept_ids = cs.get("concept_ids", [])
        link_operations = cs.get("link_operations", {})
        provenance = cs.get("provenance", {})
        pages = cs.get("pages", [])

        if not source_id:
            errors.append(f"{prefix}: source_id missing")
            continue

        if not summary_page:
            errors.append(f"{prefix}: summary_page missing")

        # outputs_created / outputs_touched disjointness
        overlap = set(outputs_created) & set(outputs_touched)
        if overlap:
            errors.append(f"{prefix}: outputs_created and outputs_touched overlap: {sorted(overlap)}")

        # summary page should usually be in outputs_created
        if summary_page and summary_page not in outputs_created:
            errors.append(f"{prefix}: summary_page not included in outputs_created")

        # pages[] ids should be unique within source
        page_ids_local = [p.get("page_id") for p in pages if p.get("page_id")]
        if len(page_ids_local) != len(set(page_ids_local)):
            dupes = sorted({x for x in page_ids_local if page_ids_local.count(x) > 1})
            errors.append(f"{prefix}: duplicate page_id within pages[]: {dupes}")

        # Every page declared in pages[] should appear in outputs_created or outputs_touched
        all_outputs = set(outputs_created) | set(outputs_touched)
        for p in pages:
            pid = p.get("page_id")
            if not pid:
                continue
            if pid not in all_outputs:
                errors.append(f"{prefix}: page_id {pid} appears in pages[] but not in outputs_created or outputs_touched")

        # Every output path should have a matching page record except maybe log/index touched externally?
        page_id_set = set(page_ids_local)
        for pid in outputs_created:
            if pid not in page_id_set:
                errors.append(f"{prefix}: outputs_created path missing matching pages[] record: {pid}")

        # For outputs_touched, allow omission only for index/log if you want looser behavior.
        # Here we enforce strictness for simplicity.
        for pid in outputs_touched:
            if pid not in page_id_set:
                errors.append(f"{prefix}: outputs_touched path missing matching pages[] record: {pid}")

        # Concept IDs should correspond to concept pages where possible
        concept_page_ids = {
            p.get("concept_id")
            for p in pages
            if p.get("page_type") == "concept" and p.get("concept_id")
        }
        for cid in concept_ids:
            if cid not in concept_page_ids:
                errors.append(f"{prefix}: concept_id '{cid}' not represented by a concept page in pages[]")

        # link_operations sanity
        for key in ("links_added", "links_removed", "backlink_edits"):
            val = link_operations.get(key)
            if not isinstance(val, int) or val < 0:
                errors.append(f"{prefix}: link_operations.{key} must be a non-negative integer")

        # provenance sanity
        for key in ("title", "parser", "compiler_version", "schema_version_used"):
            val = provenance.get(key)
            if not isinstance(val, str) or not val.strip():
                errors.append(f"{prefix}: provenance.{key} must be a non-empty string")

        # --------------------------------------------------
        # Per-page checks
        # --------------------------------------------------
        for j, p in enumerate(pages):
            pprefix = f"{prefix}.pages[{j}]"
            page_id = p.get("page_id")
            page_type = p.get("page_type")
            title = p.get("title")
            concept_id = p.get("concept_id")
            status = p.get("status")
            supports = p.get("supports_page_existence", [])
            source_refs = p.get("source_refs", [])
            outgoing_links = p.get("outgoing_links", [])
            incoming_links_known = p.get("incoming_links_known", [])
            confidence = p.get("confidence")
            orphan_candidate = p.get("orphan_candidate")

            if not page_id:
                errors.append(f"{pprefix}: missing page_id")
                continue

            if page_type and page_type not in VALID_PAGE_TYPES:
                errors.append(f"{pprefix}: invalid page_type '{page_type}'")

            inferred_type = normalize_page_type_from_path(page_id)
            if inferred_type and page_type and inferred_type != page_type:
                errors.append(
                    f"{pprefix}: page_type '{page_type}' inconsistent with page path '{page_id}' (expected '{inferred_type}')"
                )

            if not isinstance(title, str) or not title.strip():
                errors.append(f"{pprefix}: title must be a non-empty string")

            if status and status not in VALID_PAGE_STATUS:
                errors.append(f"{pprefix}: invalid status '{status}'")

            if confidence and confidence not in VALID_CONFIDENCE:
                errors.append(f"{pprefix}: invalid confidence '{confidence}'")

            if not isinstance(orphan_candidate, bool):
                errors.append(f"{pprefix}: orphan_candidate must be boolean")

            # If concept page, concept_id should exist
            if page_type == "concept" and (not isinstance(concept_id, str) or not concept_id.strip()):
                errors.append(f"{pprefix}: concept page missing concept_id")

            # If not concept page, concept_id may be null/omitted, but shouldn't be empty string
            if page_type != "concept" and concept_id == "":
                errors.append(f"{pprefix}: non-concept page has empty-string concept_id")

            # Supports should generally include current source for created/owned pages
            if page_id in outputs_created and source_id not in supports:
                errors.append(f"{pprefix}: outputs_created page does not include source_id in supports_page_existence")

            # source_refs checks
            if not isinstance(source_refs, list):
                errors.append(f"{pprefix}: source_refs must be a list")
                source_refs = []

            source_ref_ids = []
            for k, ref in enumerate(source_refs):
                rprefix = f"{pprefix}.source_refs[{k}]"
                sid = ref.get("source_id")
                role = ref.get("role")
                h = ref.get("hash")

                if not sid or not isinstance(sid, str):
                    errors.append(f"{rprefix}: source_id missing or invalid")
                else:
                    source_ref_ids.append(sid)

                if role not in VALID_SOURCE_REF_ROLES:
                    errors.append(f"{rprefix}: invalid role '{role}'")

                if "hash" in ref and (not isinstance(h, str) or not h.startswith("sha256:")):
                    errors.append(f"{rprefix}: hash must be sha256:* if present")

            # Created pages should usually have a direct source_ref to current source
            if page_id in outputs_created and source_id not in source_ref_ids:
                errors.append(f"{pprefix}: outputs_created page missing source_ref to current source_id")

            # No duplicate source_ref source_ids + role + hash triples
            seen_refs = set()
            for ref in source_refs:
                key = (ref.get("source_id"), ref.get("role"), ref.get("hash"))
                if key in seen_refs:
                    errors.append(f"{pprefix}: duplicate source_ref {key}")
                seen_refs.add(key)

            # supports_page_existence should have unique values
            if len(supports) != len(set(supports)):
                errors.append(f"{pprefix}: supports_page_existence contains duplicates")

            # If orphan_candidate=true, status should usually be orphan_candidate or stale
            if orphan_candidate and status not in {"orphan_candidate", "stale"}:
                errors.append(f"{pprefix}: orphan_candidate=true but status is '{status}'")

            # If status=orphan_candidate, orphan_candidate flag should be true
            if status == "orphan_candidate" and orphan_candidate is not True:
                errors.append(f"{pprefix}: status=orphan_candidate but orphan_candidate is not true")

            # Prevent self-links in declared known links
            if page_id in outgoing_links:
                errors.append(f"{pprefix}: page has self-link in outgoing_links")
            if page_id in incoming_links_known:
                errors.append(f"{pprefix}: page has self-link in incoming_links_known")

            # outgoing/incoming lists should be unique
            if len(outgoing_links) != len(set(outgoing_links)):
                errors.append(f"{pprefix}: outgoing_links contains duplicates")
            if len(incoming_links_known) != len(set(incoming_links_known)):
                errors.append(f"{pprefix}: incoming_links_known contains duplicates")

    # --------------------------------------------------
    # success=false behavior sanity
    # --------------------------------------------------
    if success is False and compiled_sources:
        # You may relax this later if partial success becomes supported.
        errors.append("success=false but compiled_sources is non-empty; v1 contract should keep these consistent")

    # --------------------------------------------------
    # Final result
    # --------------------------------------------------
    if errors:
        print("INVALID: compile_result.json failed semantic checks")
        for i, err in enumerate(errors, 1):
            print(f"{i}. {err}")
        sys.exit(1)

    print("OK: compile_result.json passed semantic checks")
    sys.exit(0)


if __name__ == "__main__":
    main()
```

```python name=write_compile_result.py
#!/usr/bin/env python3
"""
write_compile_result.py

Atomic writer for KDB/state/compile_result.json using:
- temp file in same directory
- flush + fsync
- os.replace
- retry/backoff for transient sync/lock issues

Intended usage:
    from compile_types import CompileResult
    from write_compile_result import write_compile_result

    result = CompileResult(...)
    write_compile_result(result)

This module assumes CompileResult has a .to_dict() method.
"""

from __future__ import annotations

import json
import os
import tempfile
import time
from pathlib import Path
from typing import Union

from compile_types import CompileResult


DEFAULT_BACKOFF = [0.1, 0.25, 0.5, 1.0, 2.0, 4.0]


def atomic_write_json(path: Path, data: dict, retries: int = 6) -> None:
    """
    Atomic JSON write with retry/backoff.

    Safe pattern:
    1. write temp file in same dir
    2. flush + fsync
    3. os.replace(temp, target)

    This helps with crash safety and transient file-lock contention
    (e.g. OneDrive/Dropbox/sync races).
    """
    path.parent.mkdir(parents=True, exist_ok=True)
    payload = json.dumps(data, indent=2, ensure_ascii=False)

    last_exc = None

    for attempt in range(retries):
        tmp_path = None
        try:
            fd, tmp_name = tempfile.mkstemp(
                prefix=f".{path.name}.tmp-",
                dir=str(path.parent),
            )
            tmp_path = Path(tmp_name)

            with os.fdopen(fd, "w", encoding="utf-8", newline="\n") as f:
                f.write(payload)
                f.flush()
                os.fsync(f.fileno())

            os.replace(tmp_path, path)
            return

        except Exception as exc:
            last_exc = exc

            if tmp_path and tmp_path.exists():
                try:
                    tmp_path.unlink()
                except Exception:
                    pass

            if attempt < retries - 1:
                delay = DEFAULT_BACKOFF[min(attempt, len(DEFAULT_BACKOFF) - 1)]
                time.sleep(delay)
            else:
                break

    raise RuntimeError(f"Failed atomic write to {path}: {last_exc}")


def write_compile_result(
    result: Union[CompileResult, dict],
    output_path: Union[str, Path] = "KDB/state/compile_result.json",
    retries: int = 6,
) -> Path:
    """
    Write compile result atomically.

    Args:
        result:
            CompileResult instance or already-serialized dict
        output_path:
            Destination JSON path
        retries:
            Number of retry attempts for transient write failures

    Returns:
        Path written to
    """
    path = Path(output_path)

    if isinstance(result, CompileResult):
        payload = result.to_dict()
    elif isinstance(result, dict):
        payload = result
    else:
        raise TypeError("result must be CompileResult or dict")

    atomic_write_json(path, payload, retries=retries)
    return path


if __name__ == "__main__":
    # Minimal demo / smoke-test payload
    demo = {
        "run_id": "demo-run",
        "success": True,
        "compiled_sources": [],
        "errors": [],
        "warnings": []
    }

    out = write_compile_result(demo)
    print(f"Wrote compile result: {out.as_posix()}")
```
